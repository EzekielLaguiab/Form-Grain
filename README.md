# 🪵 Form & Grain

> A furniture showcase website built with vanilla HTML, CSS, and JavaScript.

---

## 📖 About

**Form & Grain** is a frontend furniture website that presents a clean, modern browsing experience for furniture products. The project focuses on elegant design and smooth user interaction, built entirely with core web technologies — no frameworks required.

---

## 🛠️ Built With

| Technology | Usage |
|------------|-------|
| HTML5 | Page structure & semantic markup |
| CSS3 | Styling, layouts, and visual design |
| JavaScript | Interactivity and dynamic behavior |

---

## 📁 Project Structure

```
Form-Grain/
├── index.html      # Main HTML entry point
├── style.css       # Stylesheet for layout and design
├── script.js       # JavaScript for interactivity
└── README.md       # Project documentation
```

---

## ⚡ Getting Started

No installation or build tools needed — just clone and open!

```bash
# Clone the repository
git clone https://github.com/EzekielLaguiab/Form-Grain.git

# Navigate into the project folder
cd Form-Grain

# Open in your browser
open index.html
```

Or simply double-click `index.html` to open it in your default browser.
